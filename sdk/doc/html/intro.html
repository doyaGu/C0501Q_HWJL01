Introduction

Castor3 SDK Alpha

New Features

  • Configuration GUI
    • Kconfig
  • Preemtive OpenRTOS
  • Standard POSIX APIs
    • Easy to porting 3rd-party libraries/applications
    • Peripheral drivers using as linux drivers (open/read/ioctl)
  • Support C++ gloal constructor
  • Library dependency
    • Build the libraries only needed
  • Error Handling
    • Blue Screen
    • LCD/OSD console
  • Multi-projects
  • Interrupt/Event driven

Well-defined HAL/PAL

HAL
  • Independent with OS
  • Low-level HW drivers
    • card-detect, clock, gpio, interrupt, backlight, lcd, rtc, timer, uart, vmem, cmdq, keypad, ...
PAL
  • Full-function newlib syscalls
    • hcc-fat (use UTF-8)
  • OS related functions
    • pthread, backtrace, ...
  • Linux-like drivers
    • g-sensor, i2c, ir, keypad, lcdconsole, nand, nor, osdconsole, power, print-buf, screen, sd, touch, uart, ...
