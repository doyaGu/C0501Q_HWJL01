Package Tool

Description

This is a tool to make upgrade package. It can:

  • Pack rom code, data files of different devices.

Usage

pkgtool [options]

-h, –help
show usage info

-q, –quiet
be quiet (no status messages)

-d, –debug
debug mode

-o <file>, –output <file>
output file

-l <file>, –list <file>
list packed file content

-n <file>, –make-nor-image <file>
make packed file to NOR image

-s <size>, –size <file>
specify image size

-k <key>, –key <key>
encrypt key value; should be 0~255

-v <version>, –version <version> specify version number; maximum 7 digits

-p, –partition
force partition

-a, –no-partition
do not partition or format

-f, –format-partition
format partition

Examples

  • Pack
    • bootloader file bootloader.rom to NOR unformatted
    • image file image.rom to NOR unformatted, position 1M
    • A:\ to NAND partition0, size 256M
    • B:\ to NAND partition1, size 512M
    • encrypt key is 128 to ITE00002.pkg
      pkgtool -o ITEPKG03.pkg <br> –nor-unformatted-data0 bootloader.rom \
      –nor-unformatted-data1-pos 0x100000 \
      –nor-unformatted-data1 image.rom \
      –nand-partiton0-size 0x10000000 \
      –nand-partiton0-dir A: \
      –nand-partiton1-size 0x20000000 \
      –nand-partiton1-dir B: \
      –key 128
  • List packed file content, key is 128
    pkgtool -l ITEPKG03.pkg –key 128
  • Make packed file to NOR image, size 4M
    pkgtool -o ITEPKG03.rom -n ITEPKG03.pkg -s 0x400000
