Amplifier

This is the driver for audio amplifier.

The driver sources are put at \sdk\driver\itp\amplifier\*.c Its parent driver is itp_amplifier.c

  • Functions
    1. ioctl()
      1. ITP_IOCTL_INIT
        Inizializes this module. Will call to itpAmplifierInit().
      2. ITP_IOCTL_ENABLE
        Enables this module. Will call to itpAmplifierEnable().
      3. ITP_IOCTL_DISABLE
        Disables this module. Will call to itpAmplifierDisable().
      4. ITP_IOCTL_MUTE
        Mutes audio. Will call to itpAmplifierMute().
      5. ITP_IOCTL_UNMUTE
        Unmutes audio. Will call to itpAmplifierUnmute().
  • GPIOs
    1. CFG_GPIO_AMPLIFIER_ENABLE
      Defines the pin number of amplifier GPIO enable/disable function.
    2. CFG_GPIO_AMPLIFIER_MUTE
      Defines the pin number of amplifier GPIO mute/unmute function.
  • Configurations
    1. CFG_AMPLIFIER_ENABLE
      Determines build/link this module or not.
    2. CFG_AMPLIFIER_MODULE
      Determines which sdk\driver\itp\amplifier\itp_amp_<module>.c will be built/linked.
