uint32_t  initClock,
uint32_t  delay 
)

Initialize IIC.

Parameters
portset IIC device port.
opModeset IIC master or slave mode.
sclk_pinGPIO pin number which connect to IIC sclk pin.
data_pinGPIO pin number which connect to IIC data pin.
initClockset initialize IIC clock.
delayspecify delay time (in ms) for each GPIO operation
Returns
uint32_t_SUCCESS if succeed, error codes of uint32_t_ERROR otherwise.
Remarks
Application must call this API first when it want to use IIC API. Before mmpIicTerminate() this API will be used once only.
