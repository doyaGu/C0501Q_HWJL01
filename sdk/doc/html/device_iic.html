IIC

This is the driver for Inter-Integrated Circuit.

The driver sources are put at \sdk\driver\iic\iic.c

  • Functions
    1. ioctl()
      1. mmpIicInitialize()
        Initialize the IIC.
      2. mmpIicTerminate()
        Terminate IIC.
      3. mmpIicStart()
        IIC start condition.
      4. mmpIicStop()
        IIC stop condition.
    2. Send() and Recieve()
      1. mmpIicRecieve()
        Recieve device data packet throught IIC.
      2. mmpIicSend()
        Send device data packet throught IIC.
      3. mmpIicSendData()
        Send data to I2C slave device.
      4. mmpIicSendDataEx()
        Send data to I2C slave device.
      5. mmpIicReceiveData()
        Receive data to I2C slave device.
      6. mmpIicReceiveDataEx()
        Receive data to I2C slave device.
    3. Read() and Write() form/to buffer
      1. mmpIicSlaveRead()
        Slave read the data from the buffer.
      2. mmpIicSlaveReadWithCallback()
        Slave read the data with callback.
      3. mmpIicSlaveWrite()
        Slave write the data to the buffer.

      1. mmpIicSetClockRate()

        Get current clock.
      3. mmpIicGenStop()
        Gen stop indication command.
    5. Lock()
      1. mmpIicLockModule()
        Lock whole IIC module to prevent re-entry issue.
      2. mmpIicReleaseModule()
        Release the locked IIC module for other task usage.

        Set the slave mode.
  • GPIOs 
    GPIO has 4 modes for different functions. To use the IIC, the following pins
    should be reminded.
    1. Pin name       mode0   mode1                   mode2           mode3
      GPIO2              GPIO2   UART0 RX/IrDA_RXL       CAPCLK          IIC SDA
      GPIO3              GPIO3   UART0 TX/IrDA_TX        UART1 TX        IIC SCL
    2. Pin name       mode0   mode1                   mode2           mode3
      GPIO11             GPIO11  NDCSN2                  IIS ZWS         IIC SDA
      GPIO12             GPIO12  NDCSN3                  IIS ZDI         IIC SCL
