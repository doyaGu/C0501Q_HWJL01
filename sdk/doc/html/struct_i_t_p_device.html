Device ioctl method (controls device operating parameters).

Parameters

requestSelects the control function to be performed.

infoDevice custom data.
Returns
Upon successful completion, it shall return a value other than -1 that depends upon the device control function. Otherwise, it shall return -1 and set errno to indicate the error.
